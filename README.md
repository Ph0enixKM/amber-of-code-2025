# Advent of code 2025 in Amber

This repository contains solutions to [Advent of Code 2025](https://adventofcode.com/2025) puzzles written in [Amber](https://amber-lang.com/), a modern programming language that compiles to Bash.

## About

Advent of Code is an annual set of Christmas-themed programming challenges that can be solved in any programming language. This project uses Amber to solve the daily puzzles, demonstrating the language's capabilities for scripting and problem-solving tasks.

## Structure

Solutions are organized by day, with each day's puzzle having its own directory or file containing the implementation in Amber.
